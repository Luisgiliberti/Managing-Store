  • Classes
    Class
    Description
    ClothingItem
    Represents a piece of clothing in the inventory. It may have attributes such as size, color, and style.
    ClothingStoreImp
    Implements the functionality of a clothing store. Manages the inventory, customer transactions, and order processing.
    ClothingStore
    Represents a clothing store, which contains an inventory of clothing items and serves customers by providing and managing clothing transactions.
    Customer
    Represents a customer who interacts with the clothing store. A customer may browse items, make purchases, and have a shopping history.
    Item
    Represents a generic item that may be available in a store. It serves as a base class for more specific item types like clothing items.
    User
    Represents a user of the system. It may include both customers and store employees who interact with the clothing store application.
    MyMain
    Main class used for testing and demonstrating the functionality of the clothing store application.
